6. Sequence Diagram - Alur Lengkap (UML)
sequenceDiagram
  actor O as Owner/Admin
  actor K as Kurir
  participant FE as Front End (React)
  participant GO as Go REST + WS
  participant LV as Laravel API
  participant AI as AI Service (A2C)
  participant DB as MySQL

  Note over O,DB: 1) Autentikasi
  O->>FE: buka aplikasi, isi email + sandi
  FE->>GO: POST /api/auth/login
  GO->>DB: cek user (verifikasi bcrypt)
  DB-->>GO: data user + company
  GO-->>FE: JWT token
  FE-->>O: masuk ke dashboard

  Note over O,DB: 2) Input Pesanan
  O->>FE: tambah pesanan (penerima, menu, qty, lokasi)
  FE->>GO: POST /api/orders (Bearer JWT)
  GO->>DB: INSERT order (status = pending)
  DB-->>GO: ok (kode #1027)
  GO-->>FE: pesanan tersimpan

  Note over O,DB: 3) Dispatch AI (optimasi)
  O->>FE: pilih kurir + centang pesanan, klik Optimasi
  FE->>GO: POST /api/dispatch/optimize
  GO->>DB: ambil depot + pesanan pending
  DB-->>GO: data lokasi + demand
  GO->>AI: POST /routing/optimize (nodes, suhu)
  AI->>AI: model A2C urutkan (spoilage-aware)
  AI-->>GO: rute terurut + risiko basi
  GO-->>FE: preview rute (jarak, ETA)

  Note over O,DB: 4) Assign + Notifikasi
  O->>FE: klik Kirim ke Kurir
  FE->>GO: POST /api/dispatch/assign
  GO->>DB: simpan routes + assignments (transaksi), order=assigned
  DB-->>GO: route_id
  GO-->>K: WebSocket notif "tugas baru"
  GO-->>FE: berhasil dikirim

  Note over K,DB: 5) Eksekusi Pengantaran
  K->>LV: GET /courier/tasks
  LV->>DB: ambil rute aktif + stops
  DB-->>LV: daftar stop terurut
  LV-->>K: tampilkan tugas
  K->>GO: WS kirim GPS {lat,lng} (berkala)
  GO->>DB: upsert current_location
  GO-->>O: WS broadcast posisi kurir (Armada Live)
  K->>LV: POST /courier/.../deliver (foto PoD)
  LV->>DB: simpan bukti, status = delivered
  DB-->>LV: ok
  LV-->>K: konfirmasi tiba

  Note over O,DB: 6) Monitoring & Analitik
  O->>FE: buka Dashboard / Armada
  FE->>GO: GET /api/analytics, GET /api/fleet
  GO->>DB: agregasi penjualan + posisi
  DB-->>GO: KPI + lokasi terkini
  GO-->>FE: dashboard ter-update
  FE-->>O: lihat hasil real-time
