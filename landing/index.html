Pertanyaan Umum

Hal yang sering ditanya.

Bagaimana AI menentukan urutan antar?+

AI menilai tiap pesanan dari risiko basi (kategori + suhu lewat model Q10), tenggat, dan jarak. Makanan paling kritis & terdekat didahulukan, disusun jadi satu rute optimal per kurir.

Apakah peta benar-benar gratis?+

Ya. MapLibre + OpenStreetMap + OSRM untuk rute/ETA — tanpa biaya API Google.

Bagaimana kurir bergabung?+

Kurir daftar dengan Catering ID unik milik perusahaan, lalu menunggu persetujuan admin.

Platform apa yang didukung?+

Aplikasi kurir di Android (unduh APK langsung). Dashboard admin di web (browser desktop).
